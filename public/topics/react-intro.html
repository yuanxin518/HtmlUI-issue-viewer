⚛️ React 简介

React 是一个用于构建用户界面的 JavaScript 库，由 Meta（原 Facebook）开发和维护。

核心特点：声明式、组件化、跨平台。

声明式 UI

React 让你以声明式的方式描述 UI，当数据变化时自动高效更新视图。

function Welcome({ name }) {
  return <h1>Hello, {name}!</h1>
}

组件化

将 UI 拆分为独立、可复用的组件，每个组件管理自己的状态。

function App() {
  const [count, setCount] = useState(0)
