🎣 React Hooks

Hooks 是 React 16.8 引入的特性，让你在函数组件中使用 state 和其他 React 功能。

Hook用途
useState管理组件内部状态
useEffect处理副作用（数据请求、订阅）
useRef引用 DOM 元素或可变值
useMemo缓存计算结果
useCallback缓存函数引用
💡 规则：Hooks 只能在函数组件顶层调用，不能在条件或循环中使用。

useState 示例

const [count, setCount] = useState(0)
const increment = () => setCount(prev => prev + 1)

useEffect 示例

useEffect(() => {
  fetch('/api/data').then(setData)
}, []) // 空依赖 → 仅挂载时执行
