📘 TypeScript 基础

TypeScript 是 JavaScript 的超集，增加了静态类型系统。

基础类型

let name: string = 'Alice'
let age: number = 30
let isDone: boolean = false
let items: string[] = ['a', 'b']

接口

interface User {
  id: number
  name: string
  email?: string  // 可选
}

const user: User = { id: 1, name: 'Alice' }

泛型

function identity<T>(arg: T): T {
  return arg
}

const result = identity<string>('hello')
